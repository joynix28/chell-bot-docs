# Documentation Chell Bot

Documentation complète pour Chell Bot - Un bot Discord multifonctions.

## Développement local

Installez Mintlify CLI :

```bash
npm i -g mintlify
```

Lancez la documentation en local :

```bash
mintlify dev
```

## Déploiement

La documentation est automatiquement déployée sur Mintlify lorsque vous poussez sur la branche `main`.

## Structure

- `/commands` - Documentation des commandes
- `/guides` - Guides d'utilisation
- `/interactions` - Boutons, modals, sélecteurs
- `/advanced` - Fonctionnalités avancées (IA, etc.)
