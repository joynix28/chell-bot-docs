## Qu'est-ce qu'un modal ?

Un modal est une fenêtre popup qui s'ouvre pour collecter des informations via un formulaire.

## Utilisation dans Chell

### Création de tickets

Lorsque vous cliquez sur "Créer un ticket", un modal s'ouvre demandant :

- **Sujet** : Le titre de votre demande
- **Description** : Détails de votre problème
- **Catégorie** : Type de ticket (optionnel)

### Soumission de suggestions

```
/suggestion
```

Ouvre un modal avec :
- **Titre de la suggestion** (court)
- **Description détaillée** (jusqu'à 1000 caractères)

### Confessions anonymes

```
/confession
```

Modal anonyme pour soumettre :
- **Votre confession** (anonyme)
- **Catégorie** (optionnel)

### Formulaires personnalisés

Créez vos propres formulaires :

```
/config-formulaire create
```

Configurez :
- Jusqu'à 5 questions
- Type de réponse (court/long)
- Questions obligatoires
- Salon de réception

## Champs disponibles

### Texte court

- **Limite** : 1 ligne, 100 caractères
- **Usage** : Titre, nom, mot unique

### Paragraphe

- **Limite** : Plusieurs lignes, 1000 caractères
- **Usage** : Description, explication détaillée

## Validation

Chell valide automatiquement :
- Champs obligatoires remplis
- Longueur minimale respectée
- Format correct (si spécifié)

<Note>
  Si la validation échoue, le modal reste ouvert et affiche l'erreur.
</Note>

## Limitations Discord

- **5 champs maximum** par modal
- **45 caractères** pour le label
- **100 caractères** pour texte court
- **1000 caractères** pour paragraphe
- **Timeout 15 minutes**

## Conseils

<Tip>
  - Gardez les questions claires et concises
  - Utilisez des placeholders pour guider
  - Marquez uniquement les champs essentiels comme obligatoires
  - Testez votre formulaire avant de le publier
</Tip>
