## Invitation du bot

1. Cliquez sur le bouton **Inviter Chell** en haut de cette page
2. Sélectionnez le serveur où vous souhaitez ajouter Chell
3. Accordez les permissions demandées (Administrateur recommandé)
4. Validez l'invitation

<Warning>
  Assurez-vous d'avoir les permissions **Gérer le serveur** pour pouvoir inviter des bots.
</Warning>

## Première configuration

### 1. Vérifier le statut du bot

Une fois Chell ajouté, utilisez :

```
/botinfo
```

Cette commande affiche les informations du bot et confirme qu'il fonctionne correctement.

### 2. Configurer les logs

Pour suivre toutes les actions sur votre serveur :

```
/setup-logs
```

Sélectionnez un salon où Chell enverra les logs (créations/suppressions de salons, membres, rôles, etc.).

### 3. Configurer la modération

Activez l'antiraid et les protections de base :

```
/config-antibot activer:Oui
```

Configurez le système de warns :

```
/config-warns
```

### 4. Personnaliser l'accueil

Créez un message de bienvenue personnalisé :

```
/config-accueil
```

Vous pouvez :
- Choisir un salon
- Personnaliser le message avec des variables
- Ajouter une image de fond
- Configurer des rôles automatiques

## Problèmes courants

<AccordionGroup>
  <Accordion title="Le bot ne répond pas à mes commandes">
    - Vérifiez que le bot est en ligne (voyant vert)
    - Assurez-vous d'utiliser `/` avant la commande (commandes slash)
    - Vérifiez les permissions du bot dans le salon
  </Accordion>
  
  <Accordion title="Les commandes de modération ne fonctionnent pas">
    - Vérifiez que le rôle de Chell est au-dessus des rôles à modérer
    - Accordez la permission "Administrateur" pour un fonctionnement optimal
  </Accordion>
  
  <Accordion title="Les logs ne s'affichent pas">
    - Utilisez `/setup-logs` pour configurer le salon de logs
    - Vérifiez que Chell peut envoyer des messages dans ce salon
  </Accordion>
</AccordionGroup>
