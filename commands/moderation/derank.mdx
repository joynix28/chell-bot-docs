## Description

Retire tous les rôles d'un membre, ne laissant que le rôle @everyone. Utile pour sanctionner un membre sans le bannir.

## Utilisation

```
/derank utilisateur:[membre] raison:[raison]
```

## Paramètres

<ParamField path="utilisateur" type="utilisateur" required>
  Le membre dont retirer les rôles
</ParamField>

<ParamField path="raison" type="texte" default="Aucune raison">
  Raison du derank
</ParamField>

## Exemple

```
/derank utilisateur:@Membre raison:Abus de permissions
```

## Rôles conservés

Chell sauvegarde automatiquement les rôles retirés. Pour les restaurer :

```
/restore-roles utilisateur:@Membre
```
