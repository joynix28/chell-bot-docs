## Description

Utilise le système de **Timeout Discord** pour empêcher un membre d'envoyer des messages, de réagir ou de parler en vocal.

## Utilisation

```bash
/mute membre: @utilisateur durée: [temps] raison: [raison]
```

## Paramètres

<ParamField path="membre" type="utilisateur" required>
  Le membre à rendre muet
</ParamField>

<ParamField path="durée" type="durée" required>
  Durée du timeout (max: 28 jours)
  
  Formats acceptés : `30s`, `5m`, `2h`, `1j`, `1w`
</ParamField>

<ParamField path="raison" type="texte" default="Aucune raison fournie">
  Raison du timeout
</ParamField>

## Permissions requises

<Warning>
  **Vous :** Permission `Exclure temporairement des membres`  
  **Bot :** Permission `Exclure temporairement des membres`
</Warning>

## Limitations du timeout

<Info>
**Ce que le membre NE PEUT PAS faire :**
- ❌ Envoyer des messages dans les salons textuels
- ❌ Réagir aux messages
- ❌ Parler ou se faire entendre dans les salons vocaux
- ❌ Démarrer des activités ou threads

**Ce que le membre PEUT faire :**
- ✅ Lire les messages
- ✅ Recevoir des notifications
- ✅ Voir les salons accessibles
</Info>

## Exemples de durées

```bash
# 5 minutes
/mute membre:@Spammeur durée:5m raison:Spam

# 1 heure
/mute membre:@Toxique durée:1h raison:Insultes

# 24 heures
/mute membre:@Problématique durée:1j raison:Harcèlement

# 7 jours (max pratique)
/mute membre:@Grave durée:7j raison:Violation grave du règlement
```

## Retirer le mute

```bash
/unmute @membre
```

## Différence avec /jail

| Caractéristique | Mute (Timeout) | Jail |
|----------------|----------------|------|
| Système | Discord natif | Système bot |
| Salons visibles | Tous | Uniquement #prison |
| Rôles affectés | Aucun | Tous retirés |
| Durée max | 28 jours | Illimitée |
| Visible dans profil | Oui | Non |

<Tip>
Privilégiez `/mute` pour des sanctions rapides et temporaires. Utilisez `/jail` pour un isolement complet.
</Tip>
