## Description

Permet de bannir définitivement un membre de votre serveur Discord. Le membre ne pourra plus rejoindre le serveur tant qu'il ne sera pas débanni.

## Utilisation

```bash
/ban membre: @utilisateur raison: [raison] preuve: [image/lien]
```

## Paramètres

<ParamField path="membre" type="utilisateur" required>
  Le membre à bannir
</ParamField>

<ParamField path="raison" type="texte" default="Aucune raison fournie">
  La raison du bannissement (affichée dans les logs)
</ParamField>

<ParamField path="preuve" type="fichier/URL">
  Une preuve (capture d'écran, lien) justifiant le bannissement
</ParamField>

## Permissions requises

<Warning>
  **Vous :** Permission `Bannir des membres`  
  **Bot :** Permission `Bannir des membres`
</Warning>

## Fonctionnalités

- **Notification** : Le membre banni reçoit un message privé avec la raison
- **Logs** : Action enregistrée dans les logs de modération (si configurés)
- **Sanctions cumulatives** : Le bannissement est enregistré dans l'historique
- **Preuve** : Possibilité d'attacher une preuve (image ou lien)

## Exemples

<CodeGroup>
```bash Bannissement simple
/ban membre:@Spammeur raison:Spam répété
```

```bash Avec preuve
/ban membre:@Harceleur raison:Harcèlement preuve:https://i.imgur.com/exemple.png
```
</CodeGroup>

## Notes importantes

<Note>
- Les membres bannis ne peuvent pas rejoindre le serveur
- L'action est définitive jusqu'à débannissement
- Le membre peut faire appel via le serveur de support du bot
- Historique consultable avec `/sanctions @membre`
</Note>

## Débannir un membre

Utilisez la commande `/unban` ou allez dans Paramètres du serveur → Bannissements
