## Description

Place un membre en "prison" : il est isolé dans un salon spécifique et ne peut plus accéder au reste du serveur pendant une durée définie.

## Utilisation

```bash
/jail membre: @utilisateur durée: [temps] raison: [raison]
```

## Paramètres

<ParamField path="membre" type="utilisateur" required>
  Le membre à emprisonner
</ParamField>

<ParamField path="durée" type="durée" required>
  Durée de l'isolement (ex: `30m`, `2h`, `1j`)
</ParamField>

<ParamField path="raison" type="texte" default="Aucune raison fournie">
  Raison de la mise en prison
</ParamField>

## Permissions requises

<Warning>
  **Vous :** Permission `Gérer les rôles` ou rôle Modérateur  
  **Bot :** Permission `Gérer les rôles`
</Warning>

## Configuration préalable

<Steps>
  <Step title="Créer le salon prison">
    Créez un salon `#prison` avec permissions limitées
  </Step>
  <Step title="Configurer avec Chell">
    ```bash
    /config-prison salon:#prison role:@Prisonnier
    ```
  </Step>
  <Step title="Tester">
    Utilisez `/jail` sur un membre test
  </Step>
</Steps>

## Fonctionnement

1. Le membre reçoit le rôle **Prisonnier** (configuré)
2. Tous ses autres rôles sont **temporairement retirés**
3. Il ne peut accéder qu'au salon prison
4. À la fin du délai, ses rôles sont **automatiquement restaurés**

## Exemples

```bash
# Jail de 30 minutes
/jail membre:@Toxique durée:30m raison:Insultes

# Jail de 24 heures
/jail membre:@Perturbateur durée:1j raison:Spam massif
```

## Libération anticipée

Utilisez `/unjail @membre` pour libérer avant la fin du délai.
