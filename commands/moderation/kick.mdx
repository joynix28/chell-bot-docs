## Description

Expulse un membre du serveur. Contrairement au ban, le membre peut rejoindre à nouveau avec une invitation.

## Utilisation

```
/kick utilisateur:[membre] raison:[raison]
```

## Paramètres

<ParamField path="utilisateur" type="utilisateur" required>
  Le membre à expulser
</ParamField>

<ParamField path="raison" type="texte" default="Aucune raison fournie">
  La raison de l'expulsion (sera enregistrée dans les logs)
</ParamField>

## Permissions requises

- **Utilisateur** : Expulser des membres
- **Bot** : Expulser des membres + hiérarchie suffisante

## Exemple

```
/kick utilisateur:@Membre raison:Non-respect du règlement
```

## Différence avec /ban

| Action | Kick | Ban |
|--------|------|-----|
| Peut rejoindre | ✅ Oui | ❌ Non |
| Messages supprimés | ❌ Non | ✅ Option |
| Réversible | — | ✅ Oui |
| Sévérité | Moyenne | Élevée |
