## Description

Expulse un membre du serveur. Contrairement au ban, le membre pourra rejoindre de nouveau s'il possède un lien d'invitation.

## Utilisation

```bash
/kick membre: @utilisateur raison: [raison] preuve: [image/lien]
```

## Paramètres

<ParamField path="membre" type="utilisateur" required>
  Le membre à expulser
</ParamField>

<ParamField path="raison" type="texte" default="Aucune raison fournie">
  La raison de l'expulsion
</ParamField>

<ParamField path="preuve" type="fichier/URL">
  Une preuve justifiant l'expulsion
</ParamField>

## Permissions requises

<Warning>
  **Vous :** Permission `Expulser des membres`  
  **Bot :** Permission `Expulser des membres`
</Warning>

## Différence avec /ban

| Caractéristique | Kick | Ban |
|----------------|------|-----|
| Peut rejoindre | ✅ Oui | ❌ Non |
| Sanctions | Modéré | Sévère |
| Historique | Enregistré | Enregistré |

## Exemples

```bash
/kick membre:@Perturbateur raison:Comportement inapproprié
```
